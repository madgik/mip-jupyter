# Scratch — Your Notebooks

Use this folder for your own analysis work.

## Tips

- **Copy before editing:** Canonical examples live in `examples/`. Copy a notebook here before changing it so the originals stay intact.
- **Naming:** Use descriptive names, for example `scratch/stroke_cohort_v2.ipynb`.
- **New notebooks:** When starting fresh, create files here unless you have a specific path in mind.
